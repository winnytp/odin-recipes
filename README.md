# Odin Recipes

Part of [The Odin Project](https://www.theodinproject.com/lessons/foundations-recipes) curriculum. Learn how to write a basic recipes website with HTML.

## Learning Outcomes

After this lesson, I will have learned how to:

- Set up a [GitHub](https://github.com/) repository
- Write common `Git` commands in terminal
- Create a basic `html` website
- Practice `Git` commits in terminal
